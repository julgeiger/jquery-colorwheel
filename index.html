jQuery color wheel plugin from STDev

For more info please visit STDev website

Instructions

Hover over the wheel to view colors. Hover over the square to change saturation (left to right) and value (bottom to top) of the last hue. Click to choose a web-smart color. Click on a color box e.g. "Awesome color 1" to change an exact color.

Use the keyboard to rotate hue (if "enableKeyboardShortcuts" option is set to true):
j rotates slowly counterclockwise,
k rotates quickly clockwise, and
l rotates slowly clockwise.

Jump to a primary or secondary color (if "enableKeyboardShortcuts" option is set to true): red, green, blue, cyan, magenta, or yellow. The enter key will also confirm the current color and move to next color box.
